# Sample Excel Template for Webinar Wrapper

Create an Excel file with the following columns:

| webinar_id | webinar_name | date | time | presenter_name | presenter_email | presenter_phone | attendee_name | attendee_email | attendee_phone |
|------------|-------------|------|------|---------------|-----------------|-----------------|---------------|----------------|----------------|
| 1 | Introduction to AI | 2024-01-15 | 14:00 | John Doe | john@example.com | 12345678901 | Jane Smith | jane@example.com | 12345678902 |
| 2 | Machine Learning Basics | 2024-01-15 | 16:00 | Alice Brown | alice@example.com | 12345678903 | Bob Wilson | bob@example.com | 12345678904 |
| 3 | Data Science Workshop | 2024-01-16 | 10:00 | Charlie Davis | charlie@example.com | 12345678905 | Eve Johnson | eve@example.com | 12345678906 |

## Instructions:
1. Create an Excel file (.xlsx) with the above columns
2. Fill in your actual data following the format
3. Upload through the web interface
4. The system will parse and display the webinars
5. Schedule them on Zoom (simulated for PoC)
6. Send emails and WhatsApp notifications

## Date/Time Format:
- Date: YYYY-MM-DD (e.g., 2024-01-15)
- Time: HH:MM (e.g., 14:30)
- Phone: Include country code (e.g., 12345678901)
